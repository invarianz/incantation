# Icon Attribution

The application icon is based on the "Mage" emoji from
[Google Noto Emoji](https://github.com/googlefonts/noto-emoji)
(U+1F9D9 U+200D U+2642), modified with recolored hat/robe, removed staff,
added laptop, binary inscription, and elementary OS HIG-compliant framing.
